Basys3 FPGA — VHDL

Simple
VGA Keyboard

A modular FPGA system that accepts ASCII character input via onboard switches and push buttons, and renders text in real time on a VGA monitor. Built entirely in VHDL on the Basys3 board with a custom font ROM, VGA timing controller, and debounced input handling.

VHDL Basys3 FPGA Vivado VGA 640×480 100 MHz Clock
Project Overview

How It Works

The user sets a 7-bit ASCII code on the board's switches (SW[6:0]) and presses the Enter button to append that character to an internal display buffer. The buffer feeds a VGA controller that maps each character through a font ROM to an 8×16 pixel block, assembling a full 640×480 frame in real time.

A dedicated debounce module ensures each button press generates exactly one one-cycle pulse regardless of how long the button is held — critical at 100 MHz where a single press would otherwise register thousands of times. Save and Load buttons allow storing and recalling a message independently of the live display buffer.

System Architecture

Block Diagram

The design is split into four functional layers: input debouncing, character buffering, font rendering, and VGA signal generation. All modules are instantiated and wired in top.vhd.

Implementation Notes

Key Modules

🔘

debounce.vhd

Each button input passes through a counter that starts on a rising edge and produces a single one-cycle pulse only after the signal has been stable for 1 ms (100 000 clock cycles at 100 MHz).

DEBOUNCE_MAX = 100_000
📝

char_buffer.vhd

Maintains a 32-character active buffer and a separate saved buffer. Enter appends; Clear resets; Save copies active → saved; Load copies saved → active. Output is a flat 224-bit bus (32×7).

CHAR_NUM=32 · BYTE_NUM=7
🔤

font_rom.vhd

Stores all 128 ASCII glyphs as 8×16 bitmaps. Each '1' bit produces a white pixel; '0' produces black. The VGA controller indexes into this ROM using the character code and current scanline position.

128 glyphs · 8×16 px each
📐

buffer_line.vhd

Maps the flat character buffer to screen coordinates, handling special characters: null (0000000) ends a line, line feed (0001010) advances a row, and tab (0001001) shifts the cursor 4 columns right.

NULL · LF · TAB support
📺

vga_timer.vhd

Generates the horizontal and vertical sync pulses for 640×480 @ 60 Hz VGA. Driven by a 25 MHz clock derived from the onboard 100 MHz oscillator via Vivado's Clocking Wizard IP.

25 MHz · 640×480 · 60 Hz
🎛

vga_controller.vhd

Top-level VGA module. Instantiates font_rom, buffer_line, and vga_timer. For each pixel, looks up the character, finds the correct bit in the font ROM, and drives R/G/B to all-ones (white) or all-zeros (black).

pixel → char → glyph → RGB

library IEEE;
use IEEE.STD_LOGIC_1164.ALL;
use IEEE.NUMERIC_STD.ALL;
entity top is
  port (
    clk100  : in  std_logic;                 -- 100 MHz
    reset_n : in  std_logic;                 -- BTN_C, active-low
    SW      : in  std_logic_vector(6 downto 0);  -- 7-bit ASCII
    BTN     : in  std_logic_vector(3 downto 0);  -- {Enter, Clear, Save, Load}
    VGA_HS  : out std_logic;
    VGA_VS  : out std_logic;
    VGA_R   : out std_logic_vector(3 downto 0);
    VGA_G   : out std_logic_vector(3 downto 0);
    VGA_B   : out std_logic_vector(3 downto 0)
  );
end entity;
architecture rtl of top is
  constant LROW   : integer := 16;
  constant N_CHAR : integer := 32;
  constant BYTE   : integer := 7;
  signal btn_raw    : std_logic_vector(4 downto 0);
  signal btn_pulse  : std_logic_vector(4 downto 0);
  signal enter_p, clear_p, save_p, load_p : std_logic;
  signal rst     : std_logic := '0';
  signal str_bus : std_logic_vector(N_CHAR*BYTE-1 downto 0);
begin
  rst <= not reset_n;
  btn_raw <= rst & BTN;
  deb: entity work.debounce
    port map (clk => clk100, btn_raw => btn_raw, btn_pulse => btn_pulse);
  enter_p <= btn_pulse(0);
  clear_p <= btn_pulse(1);
  save_p  <= btn_pulse(2);
  load_p  <= btn_pulse(3);
  buf: entity work.char_buffer
    generic map (N_CHAR => N_CHAR, BYTE => BYTE)
    port map (clk => clk100, reset_n => rst, enter_p => enter_p,
              clear_p => clear_p, save_p => save_p, load_p => load_p,
              sw_in => SW, str_out => str_bus);
  vg: entity work.vga
    generic map (LROW => LROW, N_CHAR => N_CHAR, BYTE => BYTE)
    port map (clk => clk100, reset => reset_n, str => str_bus,
              hsync => VGA_HS, vsync => VGA_VS,
              r => VGA_R, g => VGA_G, b => VGA_B);
end architecture;
vga.vhd
library IEEE;
use IEEE.STD_LOGIC_1164.ALL;
use IEEE.NUMERIC_STD.ALL;
entity vga is
  generic (LROW : integer := 16; N_CHAR : integer := 32; BYTE : integer := 7);
  port (
    clk    : in  std_logic;
    reset  : in  std_logic;
    str    : in  std_logic_vector(N_CHAR*BYTE-1 downto 0);
    hsync  : out std_logic;
    vsync  : out std_logic;
    r, g, b: out std_logic_vector(3 downto 0)
  );
end entity;
architecture rtl of vga is
  signal pix25        : std_logic;
  signal row_done     : std_logic;
  signal row          : std_logic_vector(4 downto 0);
  signal col          : std_logic_vector(6 downto 0);
  signal charrow      : std_logic_vector(3 downto 0);
  signal charcol      : std_logic_vector(2 downto 0);
  signal display      : std_logic;
  signal complete_line: std_logic_vector(BYTE * LROW - 1 downto 0);
  signal cur_char     : std_logic_vector(BYTE-1 downto 0);
  signal pixel_on     : std_logic;
begin
  -- Clock divider: 100 MHz -> 25 MHz
  clkdiv: process(clk, reset)
    variable cnt: unsigned(1 downto 0) := (others => '0');
  begin
    if reset = '1' then cnt := (others => '0'); pix25 <= '0';
    elsif rising_edge(clk) then cnt := cnt + 1; pix25 <= cnt(1);
    end if;
  end process;
  lb: entity work.line_buffer
    generic map (LROW => LROW, TAB => 4, N_CHAR => N_CHAR, N_BITS => 5, BYTE => BYTE)
    port map (clk => pix25, reset => reset, str => str,
              row_done => row_done, complete_line => complete_line);
  vg_t: entity work.vga_timing
    port map (clk => pix25, reset => reset, row_done => row_done,
              row => row, col => col, charrow => charrow, charcol => charcol,
              hsync => hsync, vsync => vsync, display_on => display);
  latch_char: process(pix25)
    variable idx: integer range 0 to N_CHAR-1;
  begin
    if falling_edge(pix25) then
      if unsigned(row) < 2 and unsigned(col(3 downto 0)) < 16 then
        idx := to_integer(unsigned(col(3 downto 0))) + (to_integer(unsigned(row)) * LROW);
        if idx < N_CHAR then cur_char <= str(idx*BYTE+BYTE-1 downto idx*BYTE);
        else cur_char <= "0100000"; end if;
      else cur_char <= "0100000"; end if;
    end if;
  end process;
  font_i: entity work.char_rom
    port map (char_code => cur_char, row => charrow, col => charcol, pixel => pixel_on);
  rgb_out: process(pix25)
  begin
    if rising_edge(pix25) then
      if display = '1' and pixel_on = '1' and unsigned(row) < 2 and unsigned(col) < 16 then
        r <= "1111"; g <= "1111"; b <= "1111";
      else r <= "0000"; g <= "0000"; b <= "0000"; end if;
    end if;
  end process;
end architecture;

library IEEE;
use IEEE.STD_LOGIC_1164.ALL;
use IEEE.NUMERIC_STD.ALL;
entity vga_timing is
  port(
    clk        : in  std_logic;   -- 25 MHz
    reset      : in  std_logic;
    row_done   : out std_logic;
    row        : out std_logic_vector(4 downto 0);
    col        : out std_logic_vector(6 downto 0);
    charrow    : out std_logic_vector(3 downto 0);
    charcol    : out std_logic_vector(2 downto 0);
    hsync      : out std_logic;
    vsync      : out std_logic;
    display_on : out std_logic
  );
end entity;
architecture rtl of vga_timing is
  signal hc, vc: unsigned(9 downto 0) := (others=>'0');
begin
  sync: process(clk)
  begin
    if rising_edge(clk) then
      if reset='1' then hc <= (others=>'0'); vc <= (others=>'0');
      else
        if hc = 799 then
          hc <= (others=>'0');
          if vc = 524 then vc <= (others=>'0'); else vc <= vc + 1; end if;
        else hc <= hc + 1; end if;
      end if;
    end if;
  end process;
  hsync      <= '0' when hc>=656 and hc<752 else '1';
  vsync      <= '0' when vc>=490 and vc<492 else '1';
  display_on <= '1' when hc<640 and vc<480 else '0';
  row_done   <= '1' when hc=639 and vc(3 downto 0)="1111" else '0';
  row    <= std_logic_vector(resize(vc(8 downto 4),5));
  col    <= std_logic_vector(resize(hc(9 downto 3),7));
  charrow <= std_logic_vector(vc(3 downto 0));
  charcol <= std_logic_vector(hc(2 downto 0));
end architecture;
char_buffer.vhd
library IEEE;
use IEEE.STD_LOGIC_1164.ALL;
use IEEE.NUMERIC_STD.ALL;
entity char_buffer is
  generic (N_CHAR : integer := 32; BYTE : integer := 7);
  port (
    clk     : in  std_logic;
    reset_n : in  std_logic;        -- active-low
    enter_p : in  std_logic;
    clear_p : in  std_logic;
    save_p  : in  std_logic;
    load_p  : in  std_logic;
    sw_in   : in  std_logic_vector(BYTE-1 downto 0);
    str_out : out std_logic_vector(N_CHAR*BYTE-1 downto 0)
  );
end entity;
architecture rtl of char_buffer is
  type mem_t is array(0 to N_CHAR-1) of std_logic_vector(BYTE-1 downto 0);
  signal buff  : mem_t := (others => (others => '0'));
  signal snap  : mem_t := (others => (others => '0'));
  signal ptr   : integer range 0 to N_CHAR := 0;
  signal ptr_s : integer range 0 to N_CHAR := 0;
begin
  pack: for i in 0 to N_CHAR-1 generate
    str_out((i+1)*BYTE-1 downto i*BYTE) <= buff(i);
  end generate;
  process(clk, reset_n)
    variable temp_buff : mem_t;
  begin
    if reset_n = '0' then
      buff <= (others => (others => '0')); snap <= (others => (others => '0'));
      ptr <= 0; ptr_s <= 0;
    elsif rising_edge(clk) then
      temp_buff := (others => "0100000");  -- ASCII space
      for i in 0 to N_CHAR-1 loop
        if i < ptr then temp_buff(i) := buff(i); end if;
      end loop;
      if clear_p = '1' then
        temp_buff := (others => (others => '0')); ptr <= 0;
      elsif save_p = '1' then snap <= buff; ptr_s <= ptr;
      elsif load_p = '1' then temp_buff := snap; ptr <= ptr_s;
      elsif enter_p = '1' then
        if ptr < N_CHAR then temp_buff(ptr) := sw_in; ptr <= ptr + 1; end if;
      end if;
      buff <= temp_buff;
    end if;
  end process;
end architecture;
line_buffer.vhd
library IEEE;
use IEEE.STD_LOGIC_1164.ALL;
use IEEE.STD_LOGIC_UNSIGNED.ALL;
use IEEE.NUMERIC_STD.ALL;
entity line_buffer is
  generic (LROW: integer:=16; TAB: integer:=4; N_CHAR: integer:=32; N_BITS: integer:=5; BYTE: integer:=7);
  port (
    clk, reset    : in  std_logic;
    str           : in  std_logic_vector(N_CHAR*BYTE-1 downto 0);
    row_done      : in  std_logic;
    complete_line : out std_logic_vector(BYTE*LROW-1 downto 0) := (others => '0')
  );
end line_buffer;
architecture Behavioral of line_buffer is
  signal printed     : std_logic_vector(N_BITS-1 downto 0) := (others => '0');
  signal char_line   : std_logic_vector(BYTE*LROW-1 downto 0);
  signal col         : integer range 0 to LROW-1 := 0;
  signal line_done   : std_logic := '0';
  signal current_row : integer range 0 to 1 := 0;
begin
  process(clk)
    variable parsed_char  : std_logic_vector(BYTE-1 downto 0);
    variable parse_begin  : integer range 0 to str'left;
    variable loc          : integer range 0 to N_CHAR-1;
    variable needed_space : integer range 1 to 4;
  begin
    if rising_edge(clk) then
      if reset = '1' then
        col <= 0; printed <= (others => '0'); char_line <= (others => '0');
        complete_line <= (others => '0'); line_done <= '0'; current_row <= 0;
      elsif line_done = '0' then
        loc := to_integer(unsigned(printed)) + (current_row * LROW);
        parse_begin := loc * BYTE;
        if parse_begin + BYTE - 1 <= str'high then
          parsed_char := str(parse_begin + BYTE - 1 downto parse_begin);
          printed <= printed + 1;
        else parsed_char := "0100000"; end if;
        if parsed_char = "0000000" or parsed_char = "0001010" then
          char_line(char_line'left downto col*BYTE) <= (others => '0');
          col <= 0; line_done <= '1';
        elsif parsed_char = "0001001" then
          needed_space := TAB - (col mod TAB);
          if LROW > col + needed_space then
            char_line((col+needed_space)*BYTE-1 downto col*BYTE) <= (others => '0');
            col <= col + needed_space;
          else char_line(char_line'left downto col*BYTE) <= (others => '0');
            line_done <= '1'; col <= 0; end if;
        else
          char_line((col+1)*BYTE-1 downto col*BYTE) <= parsed_char; col <= col + 1;
        end if;
        if col >= LROW-1 or loc >= N_CHAR-1 then col <= 0; line_done <= '1'; end if;
      elsif row_done = '1' then
        complete_line <= char_line; char_line <= (others => '0');
        line_done <= '0'; col <= 0;
        if current_row = 0 then
          printed <= std_logic_vector(to_unsigned(LROW, printed'length));
          current_row <= 1;
        else printed <= (others => '0'); current_row <= 0; end if;
      end if;
    end if;
  end process;
end Behavioral;
debounce.vhd
library IEEE;
use IEEE.std_logic_1164.all;
entity debounce is
  port(
    clk       : in  std_logic;
    btn_raw   : in  std_logic_vector(4 downto 0);
    btn_pulse : out std_logic_vector(4 downto 0)
  );
end entity debounce;
architecture rtl of debounce is
  constant DEBOUNCE_MAX : integer := 100000;  -- ~1 ms at 100 MHz
  type cnt_arr_t is array (4 downto 0) of integer range 0 to DEBOUNCE_MAX;
  signal cnt        : cnt_arr_t := (others => 0);
  signal clean      : std_logic_vector(4 downto 0) := (others => '0');
  signal prev_clean : std_logic_vector(4 downto 0) := (others => '0');
begin
  process(clk)
  begin
    if rising_edge(clk) then
      for i in 0 to 4 loop
        if btn_raw(i) = clean(i) then cnt(i) <= 0;
        else
          if cnt(i) < DEBOUNCE_MAX then cnt(i) <= cnt(i) + 1;
          else clean(i) <= btn_raw(i); cnt(i) <= 0; end if;
        end if;
      end loop;
      for i in 0 to 4 loop
        if clean(i) = '1' and prev_clean(i) = '0' then btn_pulse(i) <= '1';
        else btn_pulse(i) <= '0'; end if;
      end loop;
      prev_clean <= clean;
    end if;
  end process;
end architecture rtl;
char_rom.vhd — 128 ASCII glyphs, 8×16 px each
library IEEE;
use IEEE.STD_LOGIC_1164.ALL;
use IEEE.STD_LOGIC_UNSIGNED.ALL;
use IEEE.NUMERIC_STD.ALL;
entity char_rom is
    Port (
        char_code : in  std_logic_vector(6 downto 0);  -- 7-bit ASCII
        row       : in  std_logic_vector(3 downto 0);  -- 0 to 15
        col       : in  std_logic_vector(2 downto 0);  -- 0 to 7
        pixel     : out std_logic
    );
end char_rom;
architecture Behavioral of char_rom is
  type rom_type is array (0 to 127, 0 to 15) of std_logic_vector(0 to 7);
  -- 128 characters × 16 rows × 8 bits = 16,384 bits of font data
  constant font_rom : rom_type := (
    -- Glyphs 0-31: control characters (mostly blank)
    -- Glyphs 32-126: printable ASCII
    -- Glyph 127: DEL symbol
    -- (full bitmap data in source file)
    others => (others => "00000000")
  );
begin
  pixel <= font_rom(to_integer(unsigned(char_code)),
                    to_integer(unsigned(row)))
                   (to_integer(unsigned(col)));
end Behavioral;

Proximity Warning System

A real-time distance-sensing and alerting system built from scratch on the STM32F103 (Blue Pill), with no HAL or middleware. Ultrasonic ranging, analog output, PWM buzzer, 4-bit LCD, and three-zone LED indication — all driven by direct register manipulation.

Project Overview

How It Works

Every 50 ms, a TIM4 interrupt sets a g_sample_ready flag. The main loop fires a 10 µs ultrasonic trigger pulse, then reads the echo width captured by an EXTI1 edge-detection state machine. Distance is computed from pulse duration (÷58 for cm), and the system maps it to three output domains simultaneously.

A potentiometer on ADC1_IN2 lets the user tune buzzer duty cycle at runtime (10–90%). The DAC0808 receives an 8-bit value linearly scaled to 0–200 cm, providing an analog voltage proportional to proximity. The LCD refreshes each cycle showing distance, sample period, DAC value, and duty percentage.

System Architecture

Block Diagram

All peripherals hang off the STM32F103. TIM2 provides the global microsecond timebase; TIM4 drives the 20 Hz sampling cadence; EXTI1 captures the echo edges; TIM1 generates the buzzer PWM; ADC1 reads the pot; and GPIOB bit-bangs the DAC0808's 8-bit parallel bus.

Implementation Notes

Key Subsystems

Each peripheral is initialised and serviced by a focused module. There are no blocking delays in interrupt context — the main loop polls g_sample_ready, keeping ISRs minimal.

⏱

Microsecond Timebase (TIM2)

TIM2 runs at 1 MHz (PSC=71, ARR=0xFFFF). An overflow ISR increments g_tim2_ovf. micros() reconstructs a 32-bit timestamp with a double-read loop to guard against overflow races.

PSC=71 · ARR=0xFFFF · IRQ: TIM2
📡

Echo Capture (EXTI1)

Both edges of the echo pulse are captured by a three-state machine: idle → rose → complete. Rise and fall timestamps from micros() give pulse width; results outside 100–40 000 µs are discarded.

RTSR + FTSR · PA1 · state in g_echo_state
🎛

Sample Tick (TIM4)

TIM4 fires at 20 Hz (PSC=7199, ARR=499). The ISR sets g_sample_ready and advances a beep-toggle counter so the warning zone buzzer alternates every 5 ticks (250 ms).

PSC=7199 · ARR=499 · 20 Hz
🔊

PWM Buzzer (TIM1)

TIM1 CH1 on PA8 (AF push-pull). ARR=999 gives a 2 kHz carrier at 36 MHz effective clock. CCR1 is written as a percentage of ARR+1, and the BDTR MOE bit enables the complementary output stage.

TIM1 CH1 · PWM Mode 1 · BDTR MOE
📊

DAC0808 Output

Eight GPIOB pins (non-contiguous) are mapped through DAC_PIN_MAP[8]. A single BSRR write simultaneously clears all 8 bits and sets the new value, making the update glitch-free and atomic.

BSRR atomic write · DAC_PIN_MASK=0x7C0B
🖥

HD44780 LCD (4-bit, split port)

D4–D5 live on GPIOA and D6–D7 on GPIOB, handled by nibble writes with BSRR mask-clear + set. The buzzer CCR1 is zeroed during LCD refresh to prevent interference from GPIO switching noise.

PA4–7 / PB8–9 · 4-bit mode · split port

main.c

Complete bare-metal implementation. All register addresses accessed directly via CMSIS structs. No HAL, no LL drivers — every peripheral configured from scratch.

main.c — STM32F103C8T6 
#include "stm32f10x.h"
#include <stdint.h>

/* ============================================================
 *  PIN & CONSTANT DEFINITIONS
 * ============================================================ */

/* GPIOA pins */
#define PIN_TRIG           0       /* PA0 - HC-SR04 TRIG */
#define PIN_ECHO           1       /* PA1 - HC-SR04 ECHO */
#define PIN_POT_CH         2       /* PA2 - ADC1_IN2 */
#define PIN_LCD_RS         4       /* PA4 */
#define PIN_LCD_EN         5       /* PA5 */
#define PIN_LCD_D4         6       /* PA6 */
#define PIN_LCD_D5         7       /* PA7 */

/* GPIOB pins */
#define PIN_LED_SAFE       5       /* PB5 */
#define PIN_LED_WARN       6       /* PB6 */
#define PIN_LED_DANGER     7       /* PB7 */
#define PIN_LCD_D6         8       /* PB8 */
#define PIN_LCD_D7         9       /* PB9 */

static const uint8_t DAC_PIN_MAP[8] = { 0, 1, 3, 10, 11, 12, 13, 14 };
#define DAC_PIN_MASK       0x7C0B

#define SAMPLE_PERIOD_MS   50
#define DIST_SAFE_CM       50
#define DIST_DANGER_CM     20
#define DIST_DAC_MAX_CM    200
#define DIST_MAX_CM        400

#define BUZZER_DUTY_MIN    10
#define BUZZER_DUTY_MAX    90
#define BEEP_HALF_TICKS    5

/* ============================================================
 *  GLOBAL STATE
 * ============================================================ */

volatile uint32_t g_tim2_ovf      = 0;
volatile uint8_t  g_sample_ready  = 0;
volatile uint32_t g_echo_rise_us  = 0;
volatile uint32_t g_echo_fall_us  = 0;
volatile uint8_t  g_echo_state    = 0;
volatile uint8_t  g_beep_tick     = 0;
volatile uint8_t  g_beep_on       = 0;

/* ============================================================
 *  TIMEBASE (TIM2 @ 1 MHz)
 * ============================================================ */

uint32_t micros(void)
{
    uint32_t ovf_a, ovf_b;
    uint16_t cnt;
    do {
        ovf_a = g_tim2_ovf;
        cnt   = TIM2->CNT;
        ovf_b = g_tim2_ovf;
    } while (ovf_a != ovf_b);
    return ovf_a * 65536UL + cnt;
}

void wait_us(uint16_t us)
{
    uint32_t start = micros();
    while ((micros() - start) < us) { ; }
}

void wait_ms(uint16_t ms)
{
    while (ms--) { wait_us(1000); }
}

void timer2_microsecond_init(void)
{
    RCC->APB1ENR |= (1 << 0);
    TIM2->PSC  = 72 - 1;
    TIM2->ARR  = 0xFFFF;
    TIM2->CNT  = 0;
    TIM2->SR   = 0;
    TIM2->DIER = (1 << 0);
    TIM2->CR1  = (1 << 0);
    NVIC_EnableIRQ(TIM2_IRQn);
}

void TIM2_IRQHandler(void)
{
    if (TIM2->SR & 1) {
        TIM2->SR &= ~1;
        g_tim2_ovf++;
    }
}

/* ============================================================
 *  ECHO CAPTURE (EXTI1)
 * ============================================================ */

void EXTI1_IRQHandler(void)
{
    if (!(EXTI->PR & (1 << 1))) return;
    EXTI->PR = (1 << 1);
    if (GPIOA->IDR & (1 << PIN_ECHO)) {
        g_echo_rise_us = micros();
        g_echo_state   = 1;
    } else if (g_echo_state == 1) {
        g_echo_fall_us = micros();
        g_echo_state   = 2;
    }
}

/* ============================================================
 *  MAIN LOOP
 * ============================================================ */

int main(void)
{
    gpio_init();
    timer2_microsecond_init();
    adc1_pot_init();
    buzzer_init();
    lcd_init();
    echo_exti_init();
    timer4_sampling_init();

    while (1) {
        uint32_t distance_cm;
        uint8_t  dac_value, buzzer_pct;
        uint16_t adc_raw;

        if (!g_sample_ready) continue;
        g_sample_ready = 0;

        distance_cm = ultrasonic_read_cm();
        adc_raw     = adc1_read_pot();
        buzzer_pct  = BUZZER_DUTY_MIN
                    + (uint8_t)(((BUZZER_DUTY_MAX - BUZZER_DUTY_MIN)
                               * (uint32_t)adc_raw) / 4095UL);

        {
            uint32_t d = (distance_cm > DIST_DAC_MAX_CM) ? DIST_DAC_MAX_CM : distance_cm;
            dac_value  = (uint8_t)((255UL * d) / DIST_DAC_MAX_CM);
            dac0808_write(dac_value);
        }

        apply_warning(distance_cm, buzzer_pct);
        lcd_refresh(distance_cm, dac_value, buzzer_pct);
        ultrasonic_trigger();
    }
}
